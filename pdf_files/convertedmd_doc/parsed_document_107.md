# 4.2.4 小结

试验表明，冲击次数和温度对通风率标准棒、卷烟通风率均有影响，但与温度相比，冲击次数对卷烟通风率的影响较小，温度因素对卷烟通风率的影响起决定作用。而采用通风率标准棒对仪器进行校准，可有效减弱温度对通风率标准棒和卷烟通风率测量结果的影响。
