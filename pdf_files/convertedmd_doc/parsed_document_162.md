# 单因素试验结果与讨论

（1）卷烟纸透气度对纸通风率、总通风率的影响如图 1 所示，随着卷烟纸透气度的增加，卷烟总通风率显著增加。通过回归分析，可以得到拟合模型：y=0.135x+17.314(R²=0.975)

（2）无胶区宽度与总通风率的影响如图 2 所示，可以看出当无胶区由 5mm 增加到 7mm 时，总通风率基本无增加；由 7mm 增加到 9mm 时，总通风率显著增加。

（3）自透接装纸透气度对卷烟总通风率的影响如图 3 所示，可以看出，卷烟总通风率与自透接装纸透气度呈正向线性关系，回归分析可以得到拟合模型：y=0.085x+16.899(R²=0.968)
